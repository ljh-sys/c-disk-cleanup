# C盘清理分析技能

## 概述

一键扫描 C 盘关键路径，生成只读的 HTML 分析报告，按类别分组展示所有大文件/目录，由 AI 自动匹配中文说明，帮助用户判断哪些可以清理。

## 触发条件

- "分析C盘" / "扫描C盘" / "C盘分析"
- "C盘清理" / "清理C盘" / "C盘空间"
- "磁盘清理" / "看看C盘"

## 工作流程

### 步骤1：运行扫描

```powershell
powershell -ExecutionPolicy Bypass -File "$env:USERPROFILE\.claude\skills\c-disk-cleanup\scan.ps1"
```

脚本自动完成扫描 → 生成数据 → 生成报告 → 打开浏览器。

### 步骤2：告知用户

汇报关键数据：C 盘使用率、总扫描项数、各类风险项数量和容量。

## 报告功能

生成的 HTML 只读报告包含：

- **页首**：C 盘概览（总容量 / 已用 / 剩余）、使用率进度条（颜色随比例变化：绿 < 60% / 黄 60-80% / 红 > 80%）、扫描统计
- **中/英文切换**：右上角按钮，偏好保存到 localStorage
- **侧边栏**：按类别分组的目录导航，显示每类条目数和总大小，滚动联动高亮
- **文件列表**：每个类别一张表格，4 列 — 风险（色标）| 路径 | 大小 | 说明
- **AI 说明**：根据路径关键词自动匹配中文描述（70+ 条规则），覆盖 QQ/微信、JetBrains、Docker、NVIDIA、Chrome 等常见应用
- **风险等级**：安全（绿）/ 需确认（黄）/ 危险（红）

报告不包含交互式清理功能（无复选框、筛选、搜索），用户需自行判断并手动清理。

## 文件位置

| 文件 | 路径 |
|------|------|
| 扫描脚本 | `技能目录/scan.ps1` |
| 报告生成器 | `技能目录/make-report.py` |
| 扫描数据 | `~\.cleanup\data.json` |
| HTML 报告 | `~\Desktop\C盘清理分析报告.html` |

## 注意事项

- 扫描使用 PowerShell 并行 Job，可能需要 30-90 秒
- 不要硬编码用户名，使用 `$env:USERNAME` 和 `$env:USERPROFILE`
- 白名单和危险模式在 scan.ps1 中维护
- HTML 模板嵌入在 make-report.py 中（`report-template.html` 仅作为生成缓存）
